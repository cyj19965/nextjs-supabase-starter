- [x] 개발 환경 설정 — 커맨드/훅/에이전트/MCP/docs (미션 22)

### 1-3. 핵심 기능

- [ ] 기록 작성 폼 (`/places/new`) — server action으로 insert

- [ ] 기록 상세/수정/삭제 (`/places/[id]`)
- [ ] 카테고리 필터

### Phase 1 완료 기준
